# Smart Wallet Application

Smart Wallet Application is a modern web application designed to help users manage their finances effectively. With a focus on usability and security, the app allows users to track income, expenses, and savingsâ€”all in one convenient platform. The goal is to empower users to gain insights into their financial habits and make informed decisions about budgeting and spending.

## Features

- **Intuitive Dashboard:**  
  Provides a comprehensive view of your financial health.

- **Transaction Management:**  
  Easily add, update, or delete financial transactions.

- **Expense Tracking:**  
  Visualize your spending with interactive charts and reports.

- **Budgeting Tools:**  
  Set spending limits and receive alerts when nearing your budget.

- **User Authentication:**  
  Secure registration and login functionalities.

- **Responsive Design:**  
  Ensures a seamless experience on both desktop and mobile devices.

## Technologies Used

### Backend

- **Spring Framework (Java):**  
  Provides a robust and scalable foundation for building the application.

### Frontend

- **HTML5 & CSS3:**  
  For structuring and styling the application.
- **Thymeleaf:**  
  Serves as the templating engine to dynamically generate HTML pages.
